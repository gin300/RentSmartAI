# 《中华人民共和国民法典》租赁合同核心条文速读（第703-734条）

> 说明：本资料用于租房合规科普，不构成正式法律意见。发生争议时请结合最新法条、地方法规及证据材料判断。

## 一、租赁合同的基本规则（703-709）

- **第703条**：租赁合同是出租人将租赁物交付承租人使用、收益，承租人支付租金的合同。  
- **第704条**：租赁合同主要应写明：租赁物、用途、期限、租金、支付方式、维修责任等。  
- **第705条**：租赁期限不得超过20年；超过部分无效。  
- **第706条**：租赁期限六个月以上，原则上应采用书面形式；未采用书面形式且无法确定期限的，视为不定期租赁。  
- **第707条**：承租人按约定方式支付租金。  
- **第708条**：出租人应按约定交付符合使用条件的房屋。  
- **第709条**：出租人应保证承租人在租赁期间合法、稳定使用租赁物。  

## 二、维修、使用与费用分担（710-716）

- **第710条**：出租人负有维修义务；当事人另有约定除外。  
- **第711条**：承租人应合理使用租赁物，不得擅自改变房屋主体结构。  
- **第712条**：承租人未按约定用途使用，造成损失的，应承担赔偿责任。  
- **第713条**：承租人经出租人同意可改善、增设附属物；未经同意擅自增设，出租人可要求恢复原状或赔偿。  
- **第714条**：因承租人原因导致第三人损害的，承租人可能承担侵权责任。  
- **第715条**：出租人不履行维修义务，影响使用的，承租人可自行维修，合理费用可请求出租人承担。  
- **第716条**：因不可归责于承租人的原因导致租赁物部分或全部毁损、灭失，承租人可请求减少租金或解除合同。  

## 三、转租与第三人关系（717-721）

- **第717条**：承租人转租应经出租人同意；未经同意转租，出租人可解除合同。  
- **第718条**：经同意转租后，原承租人对出租人仍负有责任，除另有约定。  
- **第719条**：转租期限不得超过承租人剩余租期，超过部分对出租人不生效。  
- **第720条**：次承租人因使用租赁物造成损失，出租人可向承租人主张责任。  
- **第721条**：承租人对次承租人的违约行为承担相应责任。  

## 四、房屋买卖不破租赁与优先承租（722-726）

- **第722条（买卖不破租赁）**：租赁物在租赁期间发生所有权变动，不影响租赁合同效力。  
- **第723条**：出租人出卖租赁房屋，应在合理期限内通知承租人。  
- **第724条**：同等条件下，承租人享有优先购买权（法律另有规定除外）。  
- **第725条**：出租人未依法通知导致承租人损失，承租人可依法主张责任。  
- **第726条**：承租人主张优先购买权应在合理期限内行使。  

## 五、期限届满、续租、返还（727-734）

- **第727条**：租赁期限届满，承租人继续使用，出租人未提出异议的，原合同继续有效，但转为不定期租赁。  
- **第728条**：不定期租赁中，双方可随时解除，但应给予合理通知期限。  
- **第729条**：租赁期满，承租人应返还租赁物，返还状态应符合约定或通常使用后的状态。  
- **第730条**：承租人逾期返还，应承担继续占有期间的使用费或损失赔偿。  
- **第731条**：出租人可请求承租人支付拖欠租金及违约责任。  
- **第732条**：承租人可请求返还多收费用或无依据扣款。  
- **第733条**：对保证金/押金处理，应依约并遵循诚信原则，不得无故扣留。  
- **第734条**：争议处理可依约协商、调解、仲裁或诉讼。  

## 六、租房实务提示

1. **合同必须写清**：租金、押金、退租条件、维修责任、违约金计算方式。  
2. **押金条款要量化**：明确“可扣项目、证据形式、结算时间（如7个工作日内）”。  
3. **交接要留证据**：入退租交接单、设备照片/视频、抄表记录、聊天记录。  
4. **警惕霸王条款**：如“任何情况押金不退”“中介单方解释权”。  
5. **发生争议先书面催告**：固定证据后再走住建投诉、消协、诉讼路径。
